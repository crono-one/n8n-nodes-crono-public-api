
# DEV Instruction for Develop Environment

## Credentials

For local testing, run n8n with your community node package and point credentials to dev:
1) In this repo, run `npm run dev` to start n8n with the node loaded.
2) In n8n, create Crono Public API credentials using dev-generated API keys.
3) Set Base URL in the credentials to `https://dev.ext.crono.one`.
